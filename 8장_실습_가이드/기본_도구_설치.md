# 기본 도구 설치

Ubuntu 22.04 기준으로 본 가이드의 실습에 필요한 도구를 설치합니다.

## 네트워크 도구

```bash
sudo apt-get install -y \
    wireshark tshark \
    linuxptp \
    ethtool \
    iproute2 \
    iperf3 \
    net-tools
```

## Python 라이브러리

```bash
pip3 install scapy paho-mqtt asyncua
```
